# CS 300: Data Structures and Algorithms

## About This Repository

This repository contains my coursework for **CS 300: Data Structures and Algorithms**. The work in this course focused on understanding how different data structures can be used to organize, search, and display information efficiently.

The main project for this course was an advising assistance program for ABCU. The program was designed to help academic advisors load course information, display a sorted course list, and search for individual courses and prerequisites.

## Course Projects

### Project One: Pseudocode and Runtime Analysis

In Project One, I created pseudocode for three different data structures:

* Vector
* Hash table
* Binary search tree

I compared each structure using runtime analysis and Big O notation. After reviewing the strengths and weaknesses of each option, I recommended using a **binary search tree** because it allows the course list to be printed in alphanumeric order using an in-order traversal.

### Project Two: Advising Assistance Program

In Project Two, I developed a command-line C++ program called `ProjectTwo.cpp`. The program allows a user to:

* Load course data from a CSV file
* Print all courses in alphanumeric order
* Search for one course and display its title and prerequisites
* Exit the program through a menu option

The program uses a binary search tree to store course objects and includes file reading, input validation, menu handling, and course lookup functionality.

## What I Learned

This course helped me better understand how data structures affect program performance. I learned that choosing the right data structure depends on what the program needs to do most often. For this project, the binary search tree was a good choice because sorted output was one of the main requirements.

I also practiced using Big O analysis to compare algorithms and data structures. This helped me think more carefully about efficiency, especially when working with searching, sorting, and inserting data.
